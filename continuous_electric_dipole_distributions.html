2 Alternative form of V V V

The equation

V ( r ) = 1 4 π ϵ o ∫ P ⋅ ԉ ^ ԉ 2 d τ ′ V(\mathbf{r})=\frac{1}{4\pi\epsilon_o}\int\frac{\mathbf{P}\boldsymbol{\cdot}\hat{\textbf{\char"0509}}}{\char"0509^2}d\tau' V(r)=4πϵo​1​∫ԉ2P⋅ԉ^​dτ′

can be written as

V ( r ) = 1 4 π ϵ o ( ∫ V ρ b ԉ d τ ′ + ∮ σ b ԉ d a ′ ) V(\mathbf{r})=\frac{1}{4\pi\epsilon_o}\left(\int_{\mathcal V} \frac{\rho_b}{\char"0509}d\tau'+\oint \frac{\sigma_b}{\char"0509}da'\right) V(r)=4πϵo​1​(∫V​ԉρb​​dτ′+∮ԉσb​​da′)

where the closed integral is taken over the closed surface of volume V \mathcal V V and

σ b ≡ P ⋅ n ^ \sigma_b\equiv \mathbf{P}\boldsymbol{\cdot} \hat{\mathbf{n}} σb​≡P⋅n^ (with P \mathbf{P} P evaluated on the surface.)

and

ρ b ≡ − ∇ ⋅ P \rho_b\equiv -\boldsymbol{\nabla}\boldsymbol{\cdot}\mathbf{P} ρb​≡−∇⋅P.

A partial proof is given in the Appendix.

Recall that for regular (non–bound) charges on a surface or in a volume, the equation for V V V is

V ( r ) = 1 4 π ϵ o ( ∫ V ρ ԉ d τ ′ + ∮ σ ԉ d a ′ ) V(\mathbf{r})=\frac{1}{4\pi\epsilon_o}\left(\int_{\mathcal V} \frac{\rho}{\char"0509}d\tau'+\oint \frac{\sigma}{\char"0509}da'\right) V(r)=4πϵo​1​(∫V​ԉρ​dτ′+∮ԉσ​da′)

This has the same form as the new equation

V ( r ) = 1 4 π ϵ o ( ∫ V ρ b ԉ d τ ′ + ∮ σ b ԉ d a ′ ) V(\mathbf{r})=\frac{1}{4\pi\epsilon_o}\left(\int_{\mathcal V} \frac{\rho_b}{\char"0509}d\tau'+\oint \frac{\sigma_b}{\char"0509}da'\right) V(r)=4πϵo​1​(∫V​ԉρb​​dτ′+∮ԉσb​​da′)

The implication is that we can use the same techniques used to find V V V or E \mathbf{E} E created by charge densities σ \sigma σ and ρ \rho ρ for problems with σ b \sigma_b σb​ and ρ b \rho_b ρb​. If we encounter a problem where ρ b \rho_b ρb​ and σ b \sigma_b σb​ (1) are on the same surfaces and (2) have the same spatial dependence as ρ \rho ρ and σ \sigma σ in a problem solved previously, we can use the previous solution with the replacement of ρ \rho ρ with ρ b \rho_b ρb​ and σ \sigma σ with σ b \sigma_b σb​.

At this point, we have essentially considered a system of two types of charges: single charges, represented by ρ f \rho_f ρf​ and pure dipoles, represented by ρ b \rho_b ρb​. There is a class of problems one can solve where you are given a fixed (or “frozen-in”) P \mathbf{P} P and are asked to find Φ \Phi Φ. In this case, one simply computes σ b \sigma_b σb​ and ρ b \rho_b ρb​ and integrates to find Φ \Phi Φ. Quite often, one does not even need to do an integration - if we know the potential for an ordinary charge distribution σ \sigma σ (or ρ \rho ρ) and we find σ b \sigma_b σb​ (or ρ \rho ρ) has the same distribution, we only need to replace σ \sigma σ (or ρ \rho ρ) with the value of σ b \sigma_b σb​ (or ρ b \rho_b ρb​) in terms of P \mathbf{P} P.

See the description for [[#Magnetized_Objects]] for a way of thinking about how an object gets polarized. The main difference is that an electric field causes the random dipoles to become aligned anti-parallel to an external electric field. If the object is somehow “frozen” so the non-random dipoles can’t randomize after the external field is turned off, you get a polarized object. Most materials are such that the induced polarization electric field tends to oppose the external electric field (dielectrics). In contrast to magnetizable objects which can enhance (paramagnets) or oppose (diamagnets) the external magnetic field.

The mnemonic I mention in class for the prefix is that “di” or “dia” means the object wants to “die” (kill) the external field. (From a language perspective, dielectrics should be called diaelectrics since dia in Greek is “oppose”, but Faraday decided it did not look right and so used “di”. If you happened to know the prefix “di” means double in Greek, you would be a bit upset about this decision; perhaps he should have used “un-di-electrics” …).
