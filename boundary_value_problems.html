1.1 Overview

Many laws of physics are expressed as either ordinary differential equations or partial differential equations.

Recall from mechanics that the equation for the position of a particle in a uniform gravitational field is given by the ordinary differential equation

m d 2 x d t 2 = − m g m\frac{d^2x}{dt^2}=-mg mdt2d2x​=−mg

This equation does not explicitly give the position of a mass versus time. To determine, one must first obtain a solution to this equation.

A general solution for x ( t ) x(t) x(t) has the form

x ( t ) = C 1 + C 2 t − g t 2 / 2 x(t)=C_1+C_2t-gt^2/2 x(t)=C1​+C2​t−gt2/2

where C 1 C_1 C1​ and C 2 C_2 C2​ are two constants.

To use this equation to find x ( t ) x(t) x(t) for a mass, the values of the two constants must be known. Usually these two constants are determined based on the initial values of x x x and d x / d t dx/dt dx/dt. For example, if x ( t = 0 ) = 0 x(t=0)=0 x(t=0)=0 and v ( t = 0 ) = 0 v(t=0)=0 v(t=0)=0, then C 1 = C 2 = 0 C_1=C_2=0 C1​=C2​=0 and the trajectory of the mass is given by x ( t ) = − g t 2 / 2 x(t)=-gt^2/2 x(t)=−gt2/2.

This type of problem is called an initial value problem – given a differential equation for the position of the mass and it’s initial values of x x x and v v v, we arrive at a solution for x x x for all t t t. Another example of an inital value problem is for the position x x x of a mass on a spring. The differential equation for x x x is d 2 x / d t 2 − ω o 2 x = 0 d^2x/dt^2-\omega_o^2x=0 d2x/dt2−ωo2​x=0. A general solution is x ( t ) = C 1 sin ⁡ ( ω o t ) + C 2 cos ⁡ ( ω o t ) x(t)=C_1\sin(\omega_o t)+C_2\cos(\omega_o t) x(t)=C1​sin(ωo​t)+C2​cos(ωo​t).

A boundary value problem involves first finding a general solution to a partial differential equation with respect to the position (instead of time). The solution will have constants whose values are determined by the conditions at the boundary. For example, in electrostatics

∂ 2 V ∂ x 2 = 0 \frac{\partial^2 V}{\partial x^2}=0 ∂x2∂2V​=0

is a partial differential equation for V V V for a system where the symmetry is such that the potential only depends on x x x. If we want to know V ( x ) V(x) V(x), we need to know a general solution to this equation. It is

V ( x ) = C 1 + C 2 x V(x)=C_1 + C_2x V(x)=C1​+C2​x

The values for C 1 C_1 C1​ and C 2 C_2 C2​ are determined based on boundary conditions. For example, V ( x = 0 ) = 0 V(x=0)=0 V(x=0)=0 and V ( x = L ) = V o V(x=L)=V_o V(x=L)=Vo​.

In this course, you will only be required to know how to derive the general solution for partial differential equations in 1–D for cartesian, cylindrical, and spherical coordinates. Derivations are given in the 1–D section.

For 2– and 2–D partial differential equations, you will be given the general solution and asked to find the unknown constants given boundary value information.

1.2 Finding V V V

There are generally two types of problems involving V V V in electrostatics:

  1. given the locations of charges, compute the electric potential and electric field at all locations in space; and

  2. given the electric potential at certain locations in space, compute the electric potential and electric field at all locations in space.

1.2.1 Type 1 Problems

For problems of type 1., we can compute V V V using

V ( r ) = ∑ i = 1 N k q i ∣ r − r i ′ ∣ V(\mathbf{r})=\sum_{i=1}^{N} \frac{kq_{i}}{|\mathbf{r}-\mathbf{r}_{i}'|} V(r)=i=1∑N​∣r−ri′​∣kqi​​

for N N N discrete charges, or the continuous approximation

V ( r ) = ∫ k d q ∣ r − r ′ ∣ V(\mathbf{r})=\int \frac{kdq}{|\mathbf{r}-\mathbf{r}'|} V(r)=∫∣r−r′∣kdq​

along with a given line, surface, or volume charge density ( λ , σ \lambda, \sigma λ,σ, or ρ \rho ρ, respectively).

In many problems, we don’t know the locations of the charges in the system. Consider the scenario shown in the following figure in which a net charge of Q Q Q is placed on a conductor.

The charges on the conductor will distribute themselves so that the electric field in the conductor is zero. In general, we don’t know what the distribution will be – we only know that whatever it is, it is such that the electric field is zero inside.

If the top cap is centered on the origin and in the x x x– y y y plane, to compute the potential due to the top cap outside of the conductor, we may be tempted to start with

V ( r ) = ∫ 0 2 π ∫ 0 R k σ ( s ′ ) ∣ r − s ′ s ^ ∣ s ′ d s ′ d ϕ ′ V(\mathbf{r})= \int_0^{2\pi}\int_0^R \frac{k\sigma(s')}{|\mathbf{r}-s'\hat{\mathbf{s}}|}s'ds'd\phi' V(r)=∫02π​∫0R​∣r−s′s^∣kσ(s′)​s′ds′dϕ′

which was the approach used to find the potential due to charges distributed on a disk with a given σ ( s ) \sigma(s) σ(s). However, in this case, we don’t know σ ( s ) \sigma(s) σ(s) and so we are stuck. A brute-force method of computing σ \sigma σ uses the fact that the V V V must be constant on the conductor: guess σ \sigma σ the top, bottom, and side surfaces, compute V V V using integration, and then repeat until a V V V is found that is constant on the conductor.

In summary, we can only use the integral formula for V V V if the charge densities are known. In general, if we place a net charge on a conductor, we do not know how it will distribute on its surface.

2.1 Cartesian

Suppose that we connect a battery with a potential difference V o V_o Vo​ to two large conducting plates. We want to know how V V V varies between the plates.

This problem can be solved using two approaches.

  1. By assuming the potential causes equal and opposite amount of charge to be uniformly distributed on the plates. This approach can only be used because we know (or assume) how the charges will distribute on the surfaces. If the plates were not large, we could not use this method.

  2. By using the boundary value method.

2.1.2 Laplace’s Equation Method

This problem may also be solved using Laplace’s equation.

Between the plates, there are no charges, so ρ = 0 \rho=0 ρ=0 and Laplace’s equation applies. If the plates are large, then the potential will only vary in the x x x–direction, in which case Laplace’s equation in cartesian components simplifies to:

∇ 2 V = ∂ 2 V ∂ x 2 = 0 \displaystyle\nabla^2V = \frac{\partial^2 V}{\partial x^2} = 0 ∇2V=∂x2∂2V​=0

This can be written as

∇ 2 V = d 2 V d x 2 = 0 \displaystyle\nabla^2V = \frac{d^2 V}{dx^2} = 0 ∇2V=dx2d2V​=0

because V V V only depends on x x x. This equation can be integrated twice to give

V ( x ) = a x + b \displaystyle V(x)=ax+b V(x)=ax+b

where a a a and b b b are constants. You can verify that V = a x + b V = ax+b V=ax+b satisfies ∂ 2 V / ∂ x 2 = 0 \partial^2 V/\partial x^2 = 0 ∂2V/∂x2=0 by differentiating it twice with respect to x x x.

The interpretation of this equation is that in a configuration where it can be argued that potential only depends on x x x, the potential must increase or decrease linearly. In this example, the two boundary conditions are

  1. V ( x = 0 ) = 0 V(x=0)=0 V(x=0)=0

  2. V ( x = d ) = V o V(x=d)=V_o V(x=d)=Vo​

These two conditions give two equations that can be solved to find the unknown constants a a a and b b b:

V ( x = 0 ) = 0 = a ⋅ 0 + b ⇒ b = 0 \displaystyle V(x=0)=0=a\cdot 0+b \Rightarrow b = 0 V(x=0)=0=a⋅0+b⇒b=0

V ( x = d ) = V o = a d + b = a d + 0 ⇒ a = V o / d \displaystyle V(x=d)=V_o=ad+b=ad+0 \Rightarrow a=V_o/d V(x=d)=Vo​=ad+b=ad+0⇒a=Vo​/d

so the solution is

V ( x ) = V o d x \displaystyle V(x) = \frac{V_o}{d}x V(x)=dVo​​x

This is the same result found using the charge method if we set V ( 0 ) = 0 V(0)=0 V(0)=0 in the charge method equation to be consistent with what was used in Laplace’s equation method.

After developing a solution to Laplace’s equation, one should always verify that the solution matches the boundary conditions, in this case by plugging in the coordinates of the boundary:

  1. V ( x ) = V o d x ⇒ V ( 0 ) = 0 V(x) = \frac{V_o}{d}x \Rightarrow V(0)=0 V(x)=dVo​​x⇒V(0)=0 so the solution matches the x = 0 x=0 x=0 boundary condition

1-D examples are simple. As will be seen when 2- and 3-D problems are considered, typically, one cannot find a single equation that satisfies all boundary conditions by following this method; a function can be found that satisfies some but not all of the boundary conditions. To fully solve the problem, one has to rely on the so-called “Fourier Trick” to find a solution. This method is described in the section on 2-D Cartesian.
