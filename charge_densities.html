1 Introduction

Charge densities are used for two types of problems in E&M.

  1. Computing E \mathbf{E} E using Coulomb’s law due to an continuous distribution of charge and

  2. Computing the charge enclosed in a Gaussian sphere.

Charges are quantized and so technically it does not make sense to discuss a continuous charge distribution. However, when there are many charges, the contiuous approximation is very accurate when used for computing an electric field.

To create a discrete charge distribution, imagine smearing discrete charges along a line or on a surface.

  • A line of length L L L with a charge Q Q Q spread uniformly on it has a charge density of λ = Q / L \lambda=Q/L λ=Q/L with units of [ C ] / [ m ] [\text{C}]/[\text{m}] [C]/[m]. If nonuniform, the notation λ ( l ) \lambda(l) λ(l) or λ ( r ) \lambda(\mathbf{r}) λ(r) is used and λ = d Q / d l \lambda=dQ/dl λ=dQ/dl is the the density at a location where d Q dQ dQ is in a length d l dl dl.

  • A surface with area A A A and a charge Q Q Q spread uniformly on it has a charge density of σ = Q / A \sigma=Q/A σ=Q/A with units of [ C ] / [ m 2 ] [\text{C}]/[\text{m}^2] [C]/[m2]. If nonuniform, the notation σ ( x , y ) \sigma(x,y) σ(x,y) or σ ( r ) \sigma(\mathbf{r}) σ(r) is used and σ = d Q / d A \sigma=dQ/dA σ=dQ/dA is density at a location where d Q dQ dQ is in an area d A dA dA.

  • A volume with volume τ \tau τ and a charge Q Q Q spread uniformly in it has a charge density of ρ = Q / τ \rho=Q/\tau ρ=Q/τ with units of [ C ] / [ m 3 ] [\text{C}]/[\text{m}^3] [C]/[m3]. If nonuniform, the notation ρ ( x , y , z ) \rho(x,y,z) ρ(x,y,z) or ρ ( r ) \rho(\mathbf{r}) ρ(r) is used and ρ = d Q / d τ \rho=dQ/d\tau ρ=dQ/dτ at a location where d Q dQ dQ is in a volume d τ d\tau dτ.

2 Linear Charge Density λ \lambda λ

2.1 Uniform

λ \lambda λ is the charge per unit length. The total charge on a line of length L L L when λ \lambda λ is uniform is Q = λ L Q=\lambda L Q=λL.

If the line is curved, integration may be needed to compute the length of the line using L = ∫ L d l L=\int_{\mathcal{L}}dl L=∫L​dl.

In order to do an integration, this generic length must be written in terms of a coordinate system (e.g., Cartesian, cylindrical, spherical). For example, if the line of charge is along the x x x-axis, then d l = d x dl=dx dl=dx. If the line of charge is along the y y y-axis, then d l = d y dl=dy dl=dy. If the line of charge is a circle of radius b b b in the x x x- y y y plane centered on the origin, then d l = d s dl=ds dl=ds, where d s ds ds is a differential length along a circle. When d l dl dl cannot be written by inspection, the following general procedure is needed.

In the following diagram, a differential element of a curve is shown. To write this in Cartesian coordinates, a right triangle is used to relate d x dx dx, d y dy dy, and d l dl dl.

( d l ) 2 = ( d x ) 2 + ( d y ) 2 ⇒ d l = ( d x ) 2 + ( d y ) 2 (dl)^2 = (dx)^2 + (dy)^2\quad\Rightarrow\quad dl = \sqrt{(dx)^2 + (dy)^2} (dl)2=(dx)2+(dy)2⇒dl=(dx)2+(dy)2​

This equation cannot yet be used for integration because it involves two differentials. Factoring out d x dx dx gives

d l = ∣ d x ∣ 1 + ( d y d x ) 2 dl=|dx|\sqrt{1 + \left(\frac{dy}{dx}\right)^2} dl=∣dx∣1+(dxdy​)2​

(The factor in the square root also appears in the equation for the unit normal vector to a line derived in Vectors.)

If we integrate in the direction of increasing x x x, then ∣ d x ∣ = d x |dx|=dx ∣dx∣=dx and so

d l = d x 1 + ( d y d x ) 2 dl=dx\sqrt{1 + \left(\frac{dy}{dx}\right)^2} dl=dx1+(dxdy​)2​

2.2 Non–uniform

When λ \lambda λ varies along a line, it is related to the total charge according to

d Q = λ ( l ) d l dQ = \lambda(l) dl dQ=λ(l)dl

where d l dl dl is a generic differential length and l l l is a position on the line. Equivalently,

λ ( l ) = d Q d l \lambda(l) = \frac{dQ}{dl} λ(l)=dldQ​

Integration of λ ( l ) d l \lambda(l)dl λ(l)dl over the length of the line gives the total charge on a line. Q = ∫ L λ ( l ) d l Q=\int_{\mathcal L} \lambda(l) dl Q=∫L​λ(l)dl

In order to do an integration, the generic length l l l variable (and d l dl dl) must be written in terms of a coordinate system (e.g., Cartesian, cylindrical, spherical).

3.1 Uniform

σ \sigma σ is the charge per unit length. The total charge on a surface is Q = λ A Q=\lambda A Q=λA, where A A A is the surface area.

Q = σ ∫ A d A Q = \sigma\int_{\mathcal{A}} dA Q=σ∫A​dA

As with lines, to perform the integral, the differential written in terms of a generic differential area d A d\mathcal{A} dA and must be expressed in a coordinate system. This is demonstrated in the following example.
