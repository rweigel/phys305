1 Introduction

For electrostatic fields, we found from Coulomb’s law that there is a scalar function such that E = − ∇ V \mathbf{E}=-\boldsymbol{\nabla}V E=−∇V. For charge distributions that are finite in extent (e.g., not an infinite line or plane of plane), we can compute E \mathbf{E} E by first computing

V ( r ) = 1 4 π ϵ o ( ∫ ρ ( r ′ ) ԉ d τ ′ + ∫ σ ( r ′ ) ԉ d A ′ + ∫ λ ( r ′ ) ԉ d l ′ ) \displaystyle V(\mathbf{r})=\frac{1}{4\pi\epsilon_o}\left(\int \frac{\rho(\mathbf{r}')}{\char"0509}d\tau' + \int \frac{\sigma(\mathbf{r}')}{\char"0509}dA' + \int \frac{\lambda(\mathbf{r}')}{\char"0509}dl'\right) V(r)=4πϵo​1​(∫ԉρ(r′)​dτ′+∫ԉσ(r′)​dA′+∫ԉλ(r′)​dl′)

and then using E = − ∇ V \mathbf{E}=-\boldsymbol{\nabla}V E=−∇V. (In most problems, only one of the charge densities in the above equation was not zero; this is the most general equation.) Given that the integrands are scalars, it is often easier to use this approach rather than computing E \mathbf{E} E directly using Coulomb’s Law:

E ( r ) = 1 4 π ϵ o ( ∫ ԉ ^ 2 ԉ 2 ρ ( r ′ ) d τ ′ + ∫ ԉ ^ 2 ԉ 2 σ ( r ′ ) d A ′ + ∫ ԉ ^ 2 ԉ 2 λ ( r ′ ) d l ′ ) \displaystyle \mathbf{E}(\mathbf{r})=\frac{1}{4\pi\epsilon_o}\left(\int \frac{\hat{\textbf{\char"0509}}\phantom{^2}}{\char"0509^2}\rho(\mathbf{r}')d\tau' + \int \frac{\hat{\textbf{\char"0509}}\phantom{^2}}{\char"0509^2}\sigma(\mathbf{r}')dA' + \int \frac{\hat{\textbf{\char"0509}}\phantom{^2}}{\char"0509^2}\lambda(\mathbf{r}')dl'\right) E(r)=4πϵo​1​(∫ԉ2ԉ^2​ρ(r′)dτ′+∫ԉ2ԉ^2​σ(r′)dA′+∫ԉ2ԉ^2​λ(r′)dl′)

The reason that we can find V V V for which E = − ∇ V \mathbf{E}=-\boldsymbol{\nabla}V E=−∇V is because electrostatic fields (field due to static charges) have the property that ∇ × E = 0 \boldsymbol{\nabla}\times \mathbf{E}=0 ∇×E=0.

We can always compute B \mathbf{B} B using the Biot–Savart law:

B ( r ) = μ o 4 π ( ∫ d l ′ × ԉ ^ 2 ԉ 2 J ( r ′ ) d τ ′ + ∫ d l ′ × ԉ ^ 2 ԉ 2 K ( r ′ ) d A ′ + ∫ d l ′ × ԉ ^ 2 ԉ 2 I ( r ′ ) d l ′ ) \displaystyle \mathbf{B}(\mathbf{r})=\frac{\mu_o}{4\pi}\left(\int \frac{d\mathbf{l}'\times\hat{\textbf{\char"0509}}\phantom{^2}}{\char"0509^2}J(\mathbf{r}')d\tau' + \int \frac{d\mathbf{l}'\times\hat{\textbf{\char"0509}}\phantom{^2}}{\char"0509^2}K(\mathbf{r}')dA' + \int \frac{d\mathbf{l}'\times\hat{\textbf{\char"0509}}\phantom{^2}}{\char"0509^2}I(\mathbf{r}')dl'\right) B(r)=4πμo​​(∫ԉ2dl′×ԉ^2​J(r′)dτ′+∫ԉ2dl′×ԉ^2​K(r′)dA′+∫ԉ2dl′×ԉ^2​I(r′)dl′)

From the Biot–Savart law, it can be shown that ∇ × B = μ o J \boldsymbol{\nabla}\times \mathbf{B}=\mu_o\mathbf{J} ∇×B=μo​J. Because ∇ × B ≠ 0 \boldsymbol{\nabla}\times \mathbf{B}\ne 0 ∇×B=0 in general, we cannot in general find a scalar function V m V_m Vm​ such that B = − ∇ V m \mathbf{B}=-\nabla V_m B=−∇Vm​.

However, we can find a vector function A \mathbf{A} A for which B = ∇ × A \mathbf{B}=\boldsymbol{\nabla}\times \mathbf{A} B=∇×A. This vector is called the “vector potential” and it applies to magnetic fields only.

A natural question is why we introduce this vector potential (see also 15-4 in the Feynman Lectures on Physics). One of the reasons that V V V is useful is that it is a scalar and so instead of discussing the vector function E \mathbf{E} E we can discuss the scalar function V V V. In addition, V V V has a physical interpretation and use – for example, it can be used to find the work required to assemble a charge distribution. The vector potential A \mathbf{A} A does not have these two advantages. There are, however, several uses for A \mathbf{A} A. One is that the equations finding it look similar to that for V V V:

A ( r ) = 1 4 π ϵ o ( ∫ J ( r ′ ) ԉ d τ ′ + ∫ K ( r ′ ) ԉ d A ′ + ∫ I ( r ′ ) ԉ d l ′ ) \displaystyle \mathbf{A}(\mathbf{r})=\frac{1}{4\pi\epsilon_o}\left(\int \frac{\mathbf{J}(\mathbf{r}')}{\char"0509}d\tau' + \int \frac{\mathbf{K}(\mathbf{r}')}{\char"0509}dA' + \int \frac{\mathbf{I}(\mathbf{r}')}{\char"0509}dl'\right) A(r)=4πϵo​1​(∫ԉJ(r′)​dτ′+∫ԉK(r′)​dA′+∫ԉI(r′)​dl′)

As a result, some of the techniques used in computing V V V can be used in computing A \mathbf{A} A. In addition, some of the integrals found for V V V can be re–used when they appear in integrals required to compute A \mathbf{A} A. Recall that we used the expansion of 1 / ԉ 1/\char"0509 1/ԉ to find the multipole expansion of V V V (so as to avoid having to do integration to find V V V at points for which r > r ′ r>r' r>r′). The integrals for A \mathbf{A} A have a 1 / ԉ 1/\char"0509 1/ԉ, and so we can use the multipole expansion to find an equation for A \mathbf{A} A that allows us to avoid having to do integration to find A \mathbf{A} A (and also B \mathbf{B} B, because it can be computed from A \mathbf{A} A).

A second reason that A \mathbf{A} A is useful is that we can write ∇ 2 A = − μ o J \nabla^2\mathbf{A}=-\mu_o\mathbf{J} ∇2A=−μo​J, which is the analog to the Poisson equation ∇ 2 V = − ρ / ϵ o \nabla^2V=-\rho/\epsilon_o ∇2V=−ρ/ϵo​. Sometimes we can use the boundary value techniques for solving Poisson’s equation for solving ∇ 2 A = − μ o J \nabla^2\mathbf{A}=-\mu_o\mathbf{J} ∇2A=−μo​J (which is actually three equations, each which have the form of Poisson’s equation: ∇ 2 A x = − μ o J x \nabla^2A_x=-\mu_oJ_x ∇2Ax​=−μo​Jx​, ∇ 2 A y = − μ o J y \nabla^2A_y=-\mu_oJ_y ∇2Ay​=−μo​Jy​, ∇ 2 A z = − μ o J z \nabla^2A_z=-\mu_oJ_z ∇2Az​=−μo​Jz​.)

Finally, the vector potential appears as a natural quantity in the Lagrangian in classical mechanics and in many equations in quantum mechanics.
