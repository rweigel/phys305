1 Definition

See 1.2.4 of Griffiths.

The divergence of a vector field is is a scalar defined to be

div ( U ) = lim ⁡ Δ τ → 0 ∮ U ⋅ d A Δ τ lim ⁡ Δ τ → 0 Φ U Δ τ \text{div}(\mathbf{U})=\lim_{\Delta \tau\rightarrow 0}\frac{\oint \mathbf{U}\boldsymbol{\cdot} d\mathbf{A}}{\Delta \tau}\lim_{\Delta \tau\rightarrow 0}\frac{\Phi_U}{\Delta \tau} div(U)=Δτ→0lim​Δτ∮U⋅dA​Δτ→0lim​ΔτΦU​​

at any point in space, where Δ τ \Delta \tau Δτ is a differential volume that surrounds the point. The numerator is a flux, so the physical interpretation is that it represents the net flux into or out of a vanishingly small volume.

In general, this formula will not be used to compute divergence. However, it is important to understand its meaning using it at least once, which is done in the following example.

Note that Griffiths does not state the definition given above. Instead, in Chapter 1.2.4, he uses a formula for div ( U ) \text{div}(\mathbf{U}) div(U) that can be derived from the definition and only states that the divergence is “a measure of how much the vector U \mathbf{U} U spreads out (diverges) from the point in question”.

One needs to be very careful with this definition as it does not always work as expected. Examples of this were given in class. There are two vector functions for each coordinate system that you should know

Cartesian

  1. U = x ^ \mathbf{U} = \hat{\mathbf{x}} U=x^ for x > 0 x>0 x>0 and U = x ^ \mathbf{U} = \hat{\mathbf{x}} U=x^ for x < 0 x<0 x<0 – the divergence is zero for all x x x except as x → 0 x \rightarrow 0 x→0 where it approaches infinity.

  2. U = x x ^ \mathbf{U} = x\hat{\mathbf{x}} U=xx^ – the divergence is constant for all x x x

Cylindrical

  1. U = s s ^ \mathbf{U}=s\hat{\mathbf{s}} U=ss^ – the divergence is constant for all s s s

  2. U = s ^ / s \mathbf{U}=\hat{\mathbf{s}}/s U=s^/s – the divergence is zero for all s s s except as s → 0 s\rightarrow 0 s→0 where it approaches infinity.

Spherical

  1. U = r r ^ \mathbf{U}=r\hat{\mathbf{r}} U=rr^ – the divergence is constant for all r r r

  2. U = r ^ / r 2 \mathbf{U}=\hat{\mathbf{r}}/r^2 U=r^/r2 – the divergence is zero for all r r r except as r → 0 r\rightarrow 0 r→0 where it approaches infinity.

Answer

A side view of a cube with sides of Δ x \Delta x Δx, Δ y \Delta y Δy, and Δ z \Delta z Δz is shown in the following figure.

1. Only two of the six sides contribute to the flux ∮ U ⋅ d A \oint \mathbf{U}\boldsymbol{\cdot} d\mathbf{A} ∮U⋅dA. div ( U ) = lim ⁡ Δ x → 0 U ( x + Δ x / 2 ) − U ( x − Δ x / 2 ) Δ x \displaystyle\text{div}(\mathbf{U})=\lim_{\Delta x\rightarrow 0}\frac{U(x+\Delta x/2)-U(x-\Delta x/2)}{\Delta x} div(U)=Δx→0lim​ΔxU(x+Δx/2)−U(x−Δx/2)​

which is the definition of the derivative of U x ( x ) U_x(x) Ux​(x). We conclude that

div ( U ) = d U x ( x ) d x \displaystyle\text{div}(\mathbf{U})=\frac{dU_x(x)}{dx} div(U)=dxdUx​(x)​

2. When U x ( x ) = 0 U_x(x)=0 Ux​(x)=0, d U x ( x ) d x = 0 \displaystyle\frac{dU_x(x)}{dx}=0 dxdUx​(x)​=0, so div ( U ) = 0 \text{div}(\mathbf{U})=0 div(U)=0. The geometrical interpretation of this result is that for a small volume the flux in equals the flux out, which should be clear from the diagram.

If ∂ U x / ∂ x \partial U_x/\partial x ∂Ux​/∂x, ∂ U y / ∂ y \partial U_y/\partial y ∂Uy​/∂y, and ∂ U z / ∂ z \partial U_z/\partial z ∂Uz​/∂z all exist, then we do not need to evaluate a surface integral and take a limit to compute div ( U ) \text{div}(\mathbf{U}) div(U). div ( U ) = ∇ ⋅ U = ∂ U x ∂ x + ∂ U y ∂ y + ∂ U z ∂ z \text{div}(\mathbf{U}) = \boldsymbol{\nabla}\boldsymbol{\cdot}\mathbf{U}=\frac{\partial U_x}{\partial x}+\frac{\partial U_y}{\partial y}+\frac{\partial U_z}{\partial z} div(U)=∇⋅U=∂x∂Ux​​+∂y∂Uy​​+∂z∂Uz​​

∇ ≡ x ^ ∂ ∂ x + y ^ ∂ ∂ y + z ^ ∂ f ∂ z \boldsymbol{\nabla} \equiv \hat{\mathbf{x}}\frac{\partial }{\partial x}+\hat{\mathbf{y}}\frac{\partial }{\partial y}+\hat{\mathbf{z}}\frac{\partial f}{\partial z} ∇≡x^∂x∂​+y^​∂y∂​+z^∂z∂f​

was used. Caution: It is easy to mix up the gradient and divergence. The gradient involves applying the vector operator ∇ \boldsymbol{\nabla} ∇ to a scalar function f ( x , y , z ) f(x,y,z) f(x,y,z), so the result is a vector:

∇ f ( x , y , z ) = ∂ f ∂ x x ^ + ∂ f ∂ y y ^ + ∂ f ∂ z z ^ \displaystyle\boldsymbol{\nabla}f(x,y,z)=\frac{\partial f}{\partial x}\hat{\mathbf{x}}+\frac{\partial f}{\partial y}\hat{\mathbf{y}}+\frac{\partial f}{\partial z}\hat{\mathbf{z}} ∇f(x,y,z)=∂x∂f​x^+∂y∂f​y^​+∂z∂f​z^

∇ ⋅ U = ∂ U x ( x , y , z ) ∂ x + ∂ U y ( x , y , z ) ∂ y + ∂ U z ( x , y , z ) ∂ z \displaystyle\boldsymbol{\nabla}\boldsymbol{\cdot}\mathbf{U}=\frac{\partial U_x(x,y,z)}{\partial x}+\frac{\partial U_y(x,y,z)}{\partial y}+\frac{\partial U_z(x,y,z)}{\partial z} ∇⋅U=∂x∂Ux​(x,y,z)​+∂y∂Uy​(x,y,z)​+∂z∂Uz​(x,y,z)​

2 The Divergence Theorem

See also 1.3.4 of Griffiths.

The divergence theorem is

∫ V div ( U ) d τ = ∮ A U ⋅ d A = Φ U \int_{\mathcal{V}} \text{div}(\mathbf{U}) d\tau=\oint_{\mathcal A} \mathbf{U}\cdot d\mathbf{A}=\Phi_U ∫V​div(U)dτ=∮A​U⋅dA=ΦU​

In general, the defining equation for divergence involving a limit will not be used and instead div ( U ) = ∇ ⋅ U \text{div}(\mathbf{U})=\boldsymbol{\nabla}\boldsymbol{\cdot}\mathbf{U} div(U)=∇⋅U is used. In this case the divergence theorem is

∫ V ( ∇ ⋅ U ) d τ = ∮ A U ⋅ d A = Φ U \int_{\mathcal{V}} (\boldsymbol{\nabla}\boldsymbol{\cdot}\mathbf{U}) d\tau=\oint_{\mathcal A} \mathbf{U}\cdot d\mathbf{A}=\Phi_U ∫V​(∇⋅U)dτ=∮A​U⋅dA=ΦU​

This is a key vector calculus theorem that is used to derive Gauss’s law. The interpretation is that if we add up all of the divergences within a volume V \mathcal{V} V, we will get the same result if we compute the flux through the closed surface A \mathcal{A} A that encloses the volume.
